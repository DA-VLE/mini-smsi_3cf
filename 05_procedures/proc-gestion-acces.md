# Procédure de gestion des accès

## 1. Objectif

Garantir que chaque utilisateur dispose uniquement des accès nécessaires à ses missions, pendant la durée nécessaire.

## 2. Périmètre

Cette procédure couvre :
- la création de comptes ;
- la modification de droits ;
- la suppression de comptes ;
- les comptes administrateurs ;
- les accès aux dossiers partagés ;
- la messagerie professionnelle.

## 3. Principes

- Un compte doit être nominatif.
- Les droits sont accordés selon le principe du moindre privilège.
- Les accès sensibles nécessitent une validation du manager.
- Les comptes administrateurs doivent être séparés des comptes utilisateurs.
- La MFA est obligatoire pour les administrateurs et recommandée pour tous les utilisateurs.

## 4. Arrivée d'un collaborateur

1. Le manager transmet la demande au support IT.
2. La demande précise : nom, prénom, service, manager, date d'arrivée, accès nécessaires.
3. Le support IT crée le compte.
4. Les droits sont attribués selon le profil métier.
5. Un mot de passe temporaire est remis de manière sécurisée.
6. L'utilisateur change son mot de passe à la première connexion.
7. La création est tracée dans le registre des demandes.

## 5. Changement de poste

1. Le manager informe le support IT.
2. Les anciens droits sont retirés.
3. Les nouveaux droits sont attribués après validation.
4. Une vérification est réalisée sur les dossiers sensibles.

## 6. Départ d'un collaborateur

1. Le manager informe le support IT avant la date de départ.
2. Le compte est désactivé le jour du départ.
3. Les sessions actives sont fermées.
4. Les accès aux applications sont supprimés.
5. Les données professionnelles sont transférées au manager si nécessaire.
6. L'action est tracée.

## 7. Revue des droits

Une revue des droits est réalisée tous les trimestres pour :
- les dossiers RH ;
- les dossiers finance ;
- les comptes administrateurs ;
- les comptes inactifs.

## 8. Preuves attendues

- Ticket de demande d'accès.
- Validation du manager.
- Export des groupes utilisateurs.
- Capture ou journal de désactivation de compte.
- Compte rendu de revue des droits.
