# Sources et inspirations

Ce projet est une simulation pédagogique. Les documents ne remplacent pas un audit réel.

## Sources consultées / inspirations

- ISO/IEC 27001:2022 - page officielle ISO : https://www.iso.org/standard/27001
- ANSSI / MesServicesCyber - recommandations MFA et mots de passe : https://messervices.cyber.gouv.fr/guides/recommandations-relatives-lauthentification-multifacteur-et-aux-mots-de-passe
- ANSSI / MesServicesCyber - sauvegarde des systèmes d'information : https://messervices.cyber.gouv.fr/guides/fondamentaux-sauvegarde-systemes-dinformation
- ANSSI / MesServicesCyber - attaques par rançongiciels : https://messervices.cyber.gouv.fr/guides/attaques-par-rancongiciels-tous-concernes
- archoad/evalsmsi - outils d'audit et de suivi du SMSI selon ISO 27001 : https://github.com/archoad/evalsmsi

## Utilisation dans le projet

- ISO 27001 est utilisé comme inspiration pour la logique SMSI et l'approche par les risques.
- Les ressources ANSSI sont utilisées comme inspiration pour MFA, mots de passe, sauvegardes et ransomware.
- Le dépôt evalSMSI est utilisé comme inspiration générale pour la logique d'audit et de suivi SMSI.
